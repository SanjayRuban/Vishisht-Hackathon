Project Title


Simple Voting System in C


Project Objective


Develop a console-based Simple Voting System in C that allows users to cast votes for a set of predefined candidates. The system should securely record votes, display real-time vote counts, and announce the winner at the end. This project aims to provide a straightforward and efficient solution for single-instance voting scenarios.



Project Requirements


1)User Interface (UI) :


  • Design a console-based UI to guide users through the voting process.
  • Design a console-based UI to guide users through the voting process.
  • Provide feedback for successful votes and an option to view current results.

2)Voting Process :


  • Allow each user to vote once per candidate by entering a corresponding code or number.
  • Validate user input to ensure votes are counted accurately, avoiding any duplicate or invalid entries.

3)Vote Recording and Tallying :


  • Store votes securely and ensure they are accurately counted.
  • Implement a function to tally the votes in real-time, showing each candidate's vote count.

4)Result Display :


  • Provide an option to view the current vote count during the process.
  • At the end, display the final vote count for each candidate and announce the candidate with the highest votes as the winner.

5)Additional/Optional Features :


  • Add a feature to reset the voting system for a new election cycle.
  • Implement an admin mode with password protection to access and reset results.


Deliverables


  • Source code files with proper file structure.
  • A README file with instructions on how to compile and run the program.
  • Documentation outlining code structure, functions used, and any additional features added.
