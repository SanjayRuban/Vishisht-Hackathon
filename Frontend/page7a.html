Project Details

Project Title


Tic-Tac-Toe Game in C


Project Objective


Create a simple command-line based Tic-Tac-Toe game in C. The game should allow two players to play on the same computer, using the console interface. The game should include basic functionalities such as input validation, displaying the game board, checking win conditions, and providing the option to play again.



Project Requirements


1)Game Interface :


  • Display a 3x3 grid representing the Tic-Tac-Toe board.
  • Use "X" and "O" as the two players' symbols.
  • Allow players to take turns to place their symbol in an empty cell of the board.
  • After every move, update and display the current state of the board.

2)Gameplay Logic :


  • Implement player turns, allowing each player to choose an empty cell.
  • Check for winning conditions after each move (3 markers in a row, column, or diagonal).
  • Include a check for a tie if all cells are filled and no player has won.

3)Game Flow :


  • Start with a welcome screen explaining the game rules briefly.
  • Provide options to restart the game or quit after each round.
  • Ensure proper error handling for invalid moves (e.g., selecting an occupied cell or inputting incorrect data).

4)Programming Requirement :


  • Code must be written in C language and organized into functions for clarity and reusability.
  • Use comments to explain major parts of the code and logic.
  • Avoid the use of external libraries (stick to standard libraries available in C).

5)Additional Features :


  • Option for the player to play against a basic AI (computer opponent).
  • A scoreboard to track the number of wins, losses, and ties.


Deliverables


  • Source code files with proper file structure.
  • A README file with instructions on how to compile and run the program.
  • Documentation outlining code structure, functions used, and any additional features added.
