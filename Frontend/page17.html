Assignment Page

Topics Covered: "Types of Operators in C" and "Operator Precedence"

  1. What will be the output of the expression 5 + 3 * 2?
    a) 16
    b) 11
    c) 13
  2. Which of the following correctly uses a ternary operator?
    a) x ? y : z;
    b) x ? : z : y;
    c) x < y : z;
  3. In C, what type of operator is sizeof?
    a) Logical
    b) Bitwise
    c) Unary
  4. What is the result of the following expression if int x = 4; and int y = 3;: x & y?
    a) 4
    b) 0
    c) 1
  5. Which operator has the highest precedence in C?
    a) + (Addition)
    b) * (Multiplication)
    c) () (Parentheses)
  6. In the expression a + b * c - d / e, which operation is performed first?
    a) a + b
    b) d / e
    c) b * c
  7. In C, which of the following operators has the lowest precedence?
    a) &&
    b) =
    c) +
  8. For the expression a = b = c = 10;, how does operator precedence affect evaluation?
    a) c = 10 is evaluated last
    b) a = b = c is evaluated first
    c) c = 10 is evaluated first
